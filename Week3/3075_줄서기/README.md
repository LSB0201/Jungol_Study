# [JUNGOL] 3075 : 줄서기

## 📝 문제 설명
1부터 $N$까지의 번호가 적힌 카드를 하나씩 가진 $N$명의 학생들이 일렬로 서 있습니다. 각 학생들로부터 **자신보다 뒤에 서 있으면서 더 작은 번호의 카드를 가진 학생들의 명단**을 빠짐없이 받았습니다. 

순서쌍 $(X, Y)$는 학생 $Y$가 학생 $X$보다 뒤에 있으면서($X < Y$), 더 작은 카드 번호를 가지고 있다는 것을 의미합니다. 이 명단이 '하나도 빠짐없이 모두' 주어졌을 때, 각 학생들이 가진 카드 번호를 서 있는 순서대로 구하는 프로그램을 작성하시오. 

만약 주어진 명단에 모순이 있거나 올바른 카드 번호를 추해낼 수 없는 잘못된 명단이라면 `-1`을 출력해야 합니다.

---

### 📥 입력 형식
* 첫째 줄에 학생 수 $N$($1 \le N \le 100,000$)과 순서쌍의 수 $M$($0 \le M \le 1,000,000$)이 공백으로 분리되어 주어집니다.
* 다음 $M$개의 줄에는 두 자연수 $X$와 $Y$가 공백으로 분리되어 주어집니다. 이는 학생 $Y$가 학생 $X$보다 뒤에 있으면서 더 작은 카드 번호를 가졌음을 의미합니다. ($1 \le X < Y \le N$, 중복 입력 없음)

### 📤 출력 형식
* 주어진 순서쌍을 통해 카드 번호를 정확히 알아낼 수 있다면, 1번 학생부터 $N$번 학생까지의 카드 번호를 공백으로 분리하여 한 줄에 출력합니다.
* 모순이 있거나 올바른 카드 번호 배치를 찾을 수 없다면 **-1**을 출력합니다.

---

## 💻 입출력 예제

| 입력 예시 (Input) | 출력 예시 (Output) |
| :--- | :--- |
| 5 5<br>1 2<br>1 5<br>3 4<br>3 5<br>4 5 | 3 1 5 4 2 |
| 5 6<br>1 2<br>1 3<br>1 5<br>2 5<br>3 4<br>3 5 | -1 |

> **힌트/참고 사항 (필요 시)**
> * 각 학생 $X$에 대해 자신보다 뒤에 있으면서 작은 값을 가진 학생의 수(진입 차수 또는 Inversion 관계)를 이용하여 앞에서부터 혹은 뒤에서부터 유추해 나가는 세그먼트 트리(Segment Tree)나 펜윅 트리(Fenwick Tree), 혹은 세심한 정렬 조건 체크가 필요한 수리적 탐색 문제입니다.
