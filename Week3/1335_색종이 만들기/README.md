# [JUNGOL] 1335 : 색종이 만들기(영역구분)

## 📝 문제 설명
정사각형 모양의 종이가 주어지고, 각 칸은 하얀색(0) 또는 파란색(1)으로 칠해져 있습니다. 이 종이를 일정한 규칙에 따라 잘라서 모두 같은 색으로 이루어진 색종이를 만들려고 합니다.

<img width="257" height="268" alt="Image" src="https://github.com/user-attachments/assets/a299112d-6117-45fa-bb9c-dbba00bb4c72" />

종이를 자르는 규칙은 다음과 같습니다:
1. 전체 종이가 모두 같은 색으로 칠해져 있지 않으면, 가로와 세로의 중간을 잘라 똑같은 크기의 네 개의 정사각형(N/2 × N/2)으로 나눕니다.
2. 나누어진 4개의 구역 각각에 대해서도 모두 같은 색이 될 때까지 이 과정을 반복합니다.
3. 잘라진 종이가 모두 하얀색이거나, 모두 파란색이거나, 또는 하나의 칸이 되면 더 이상 자르지 않습니다.

<img width="616" height="612" alt="Image" src="https://github.com/user-attachments/assets/32baac75-876b-4053-8710-9992d36942e7" />

최종적으로 잘라진 하얀색 색종이와 파란색 색종이의 개수를 구하는 프로그램을 작성하시오.

---

### 📥 입력 형식
* 첫째 줄에는 전체 종이의 한 변의 길이 N이 주어집니다. (N은 2, 4, 8, 16, 32, 64, 128 중 하나)
* 둘째 줄부터 N개의 줄에 걸쳐 색종이의 색상 정보가 주어집니다. 하얀색은 0, 파란색은 1로 주어지며 각 숫자 사이에는 공백이 있습니다.

### 📤 출력 형식
* 첫째 줄에는 최종적으로 잘라진 하얀색 색종이의 개수를 출력합니다.
* 둘째 줄에는 최종적으로 잘라진 파란색 색종이의 개수를 출력합니다.

---

> **힌트/참고 사항 (필요 시)**
