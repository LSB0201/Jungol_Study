# [JUNGOL] 1092 : 제곱수5 (제곱수 출력)

## 📝 문제 설명
두 정수 $X$와 $Y$가 주어질 때, $X^Y$의 값을 계산하는 프로그램을 작성하시오. 
결과 값이 매우 커질 수 있으므로, 최종 결과 값을 **20091024**로 나눈 나머지를 출력해야 합니다.

---

### 📥 입력 형식
* 첫 번째 줄에 두 정수 $X$와 $Y$가 공백을 사이에 두고 주어진다.
* $X$와 $Y$는 모두 $0$ 이상 $2^{31}-1$ 이하의 정수이다.

### 📤 출력 형식
* $X^Y$를 $20,091,024$로 나눈 나머지를 출력한다.
* 단, $0^0$은 편의상 $1$로 정의한다.

---

## 💻 입출력 예제

| 입력 예시 (Input) | 출력 예시 (Output) |
| :--- | :--- |
| 2 10 | 1024 |
| 987654321 10248 | 18135873 |

---

## 🛠️ 구조 및 흐름 분석 (학습 가이드)

이 문제의 핵심은 지수 $Y$의 범위가 최대 **2,147,483,647**이라는 점입니다. $X$를 $Y$번 하나씩 곱하는 $O(Y)$ 방식은 무조건 제한 시간(1초)을 초과하므로, 지수를 절반씩 쪼개어 나가는 **분할 정복(Divide and Conquer)** 기술이 필요합니다.

### 1. 거듭제곱의 분할 정복 원리
$X^Y$를 구할 때, $Y$가 짝수인지 홀수인지에 따라 문제를 절반 크기로 압축할 수 있습니다.

* **$Y$가 짝수일 때:** $X^Y = X^{Y/2} \times X^{Y/2}$
* **$Y$가 홀수일 때:** $X^Y = X^{Y/2} \times X^{Y/2} \times X$



예를 들어 $X^{10}$을 구한다면 다음과 같이 단 4번의 계산만으로 도달할 수 있습니다.
* $X^{10} = X^5 \times X^5$
* $X^5 = X^2 \times X^2 \times X$
* $X^2 = X^1 \times X^1$
* $X^1 = X^0 \times X^0 \times X$

이 방식을 사용하면 시간 복잡도가 $O(Y)$에서 **$O(\log Y)$**로 획기적으로 줄어들어, 21억 번의 연산이 단 30번 안팎의 연산으로 끝납니다.

### 2. 나머지 연산(Modular Arithmetic)의 성질
값이 오버플로우(Overflow)되는 것을 막기 위해, 곱셈을 할 때마다 수시로 나누기 연산을 적용해야 합니다.
* $(A \times B) \pmod M = ((A \pmod M) \times (B \pmod M)) \pmod M$
