# [JUNGOL] 4181 : 순열정복 5

## 📝 문제 설명
1부터 $N$까지의 숫자를 중복을 허용하여 $M$개를 나열하되, 나열한 수들의 합이 정확히 $S$가 되는 모든 경우를 구하는 프로그램을 작성하시오.

---

### 📥 입력 형식
첫 줄에 자연수 $N, M, S$가 공백을 사이에 두고 주어진다.
* $1 \le N \le 6$
* $1 \le M \le 6$
* $1 \le S \le N \times M$

### 📤 출력 형식
문제의 조건에 맞는 모든 경우를 **오름차순**으로 출력한다. 각 숫자는 공백으로 구분하며, 각 경우는 줄바꿈으로 구분한다.

---

## 💻 입출력 예제

| 입력 예시 (Input) | 출력 예시 (Output) |
| :--- | :--- |
| 4 3 9 | 1 4 4<br>2 3 4<br>2 4 3<br>3 2 4<br>3 3 3<br>3 4 2<br>4 1 4<br>4 2 3<br>4 3 2<br>4 4 1 |
| 5 2 5 | 1 4<br>2 3<br>3 2<br>4 1 |

> **힌트/참고 사항**
> * 중복 순열을 생성하는 재귀 호출(DFS) 과정에서 현재까지 선택한 숫자들의 합을 함께 추적하면 효율적으로 해결할 수 있습니다.
> * $M$개의 숫자를 모두 고른 시점에 누적 합이 $S$와 일치하는 경우에만 출력합니다.
