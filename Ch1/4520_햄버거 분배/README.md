# [JUNGOL] 4520 : 햄버거 분배

## 📝 문제 설명
기다란 벤치 모양의 식탁에 사람들과 햄버거가 단위 간격으로 놓여 있습니다. 사람들은 자신의 위치에서 거리가 $k$ 이하인 햄버거를 왼쪽, 오른쪽 상관없이 먹을 수 있습니다.

<img width="658" height="96" alt="Image" src="https://github.com/user-attachments/assets/4355cf0c-6ab4-48f1-be80-ddcc76c9c136" />

식탁의 길이 $N$, 햄버거를 선택할 수 있는 거리 $k$, 그리고 사람과 햄버거의 위치 정보가 주어졌을 때, 햄버거를 먹을 수 있는 사람의 최대 수를 구하는 프로그램을 작성하시오.

---

### 📥 입력 형식
* 첫째 줄에 두 정수 $N$과 $k$가 공백으로 구분되어 주어집니다. ($1 \le N \le 20,000$, $1 \le k \le 10$)
* 둘째 줄에 사람과 햄버거의 위치가 문자 `P`(사람)와 `H`(햄버거)로 이루어진 길이 $N$인 문자열로 주어집니다.

### 📤 출력 형식
* 햄버거를 먹을 수 있는 최대 사람 수를 하나의 정수로 출력합니다.

---

## 💻 입출력 예제

| 입력 예시 (Input) | 출력 예시 (Output) |
| :--- | :--- |
| **예제 1**<br>20 1<br>HHPHPPHHPPHPPPHPHPHP | 8 |
| **예제 2**<br>20 2<br>HHHHHPPPPPHPHPHPHHHP | 7 |

---

## 📊 서브태스크 (Subtask)

| 번호 | 배점 | 제한 조건 |
| :--- | :--- | :--- |
| #1 | 20점 | $N \le 20$ |
| #2 | 20점 | $N \le 2,000$ |
| #3 | 60점 | 추가적인 제약 조건이 없다. |

> **💡 접근 힌트 (Greedy)**
> 최대한 많은 사람에게 햄버거를 나누어 주려면, 왼쪽부터 순서대로 사람(`P`)을 탐색하면서 그 사람이 먹을 수 있는 범위($-k \sim +k$) 내의 햄버거 중 **가장 왼쪽에 있는 햄버거(`H`)**를 우선적으로 선택하여 매칭해야 합니다. 
> 왼쪽에 있는 햄버거를 먼저 소비해 주어야 뒤에 오는 다른 사람들에게 햄버거를 선택할 수 있는 기회가 더 많이 남기 때문입니다.
