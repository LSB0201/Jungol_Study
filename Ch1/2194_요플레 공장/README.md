# [JUNGOL] 2194 : 요플레 공장

## 📝 문제 설명
요플레 공장을 운영하시는 영휘의 아버지가 어머니와 함께 여행을 가신다. 그래서 영휘는 아버지를 대신해서 요플레 공장을 $N$주 동안 운영해야 한다.
 
영휘가 걱정스러운 아버지는 영휘를 위해 $N$주 동안 필요한 우유의 양과 변동되는 가격을 미리 알아봐주었다. 아버지가 주신 정보에 의하면 각 주마다 우유는 $1ℓ$당 $C$원에 살 수 있고, 필요한 우유는 $Yℓ$이다. 
또한 우유를 $1$주일 보관하는데 우유 $1ℓ$당 $S$원의 비용이 든다. 우유는 $N$주 동안 보관이 가능하며 우유 창고가 굉장히 커서 우유를 얼마든지 보관이 가능하다.
 
영휘는 아버지에게 칭찬을 받기 위해 $N$주 동안 최소의 비용으로 요플레 공장을 운영하고 싶어서 당신에게 도움을 청했다. 당신은 영휘에게 $N$주 동안 요플레 공장을 운영할 수 있는 최소비용을 구하여 주자.

---

### 📥 입력 형식
첫 줄에 $N$과 $S$가 공백으로 구분되어 주어진다. ($1 \le N \le 100,000$, $1 \le S \le 100$)

둘째 줄부터 $N+1$번째 줄 까지 각 주의 $C[i]$와 $Y[i]$가 주어진다. ($1 \le C[i] \le 5,000$, $0 \le Y[i] \le 10,000$)
각 주는 1주부터 순서대로 입력이 된다.

### 📤 출력 형식
$N$주 동안 요플레 공장을 운영할 수 있는 최소비용을 출력한다. 
**답이 매우 클 수 있으니 유의하여라.** (자료형 주의)

---

## 💻 입출력 예제

| 입력 예시 (Input) | 출력 예시 (Output) |
| :--- | :--- |
| 4 5<br>88 200<br>89 400<br>97 300<br>91 500 | 126900 |

> **💡 주의 사항**
> 출력해야 하는 최소비용의 총합이 매우 커질 수 있으므로, C++에서 정수형 변수를 선언할 때 `int` 대신 `long long` 타입을 사용해야 오버플로우(Overflow)를 방지할 수 있습니다.
