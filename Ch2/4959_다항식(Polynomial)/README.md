# [JUNGOL] 4959 : 다항식(Polynomial)

## 📝 문제 설명
수학에서 다항식(多項식, polynomial)은 한 개 또는 두 개 이상의 항의 합으로 이루어진 식입니다. 각 항은 숫자(계수)와 문자(차수)의 곱으로 이루어집니다. 

$N$개의 다항식 정보가 주어질 때, 이 다항식들을 모두 더한 결과를 구하고 문제에서 제시한 예외 처리 규칙에 맞게 결과를 출력하는 프로그램을 작성하시오.

---

### 📥 입력 형식
* 첫 번째 줄에 다항식의 개수 $N$이 주어진다. ($2 \le N \le 50$)
* 두 번째 줄부터 $N$개의 다항식의 정보가 주어진다.
  * 각 다항식의 정보는 첫 줄에 다항식의 항 개수인 $M$이 주어진다. ($1 \le M \le 9$)
  * 이어서 $M$개의 줄에 걸쳐 계수 $A$와 차수 $B$가 공백을 사이에 두고 주어진다. 
  * ($-100 \le A \le 100$, $0 \le B \le 4$, 단 $A \neq 0$)

### 📤 출력 형식
계수가 $0$이 아닌 항들을 차수가 높은 순서(내림차순)로 다항식을 출력한다. 단, 출력 시 아래의 **예외 처리 규칙**을 반드시 준수해야 한다.

1. 최종 결과가 $0$인 경우(모든 항의 계수가 $0$이 된 경우) **`0`**을 출력한다.
2. 계수가 `1` 또는 `-1`인 경우:
   * 차수가 1 이상이면 계수의 숫자 `1`은 생략한다. (예: `1X^3` $\rightarrow$ `X^3`, `-1X^2` $\rightarrow$ `-X^2`)
   * 차수가 0(상수항)이면 숫자 `1` 또는 `-1`을 그대로 출력한다.
3. 차수가 `1`인 경우, 지수 표시(`^1`)를 제외하고 문자의 차수만 출력한다. (예: `5X^1` $\rightarrow$ `5X`)
4. 차수가 `0`인 경우, 문자 없이 계수(상수)만 출력한다.
5. 첫 번째로 출력되는 최고차항을 제외하고, 양수 계수를 가진 항 앞에는 **`+`** 부호를 붙여서 항들을 연결한다.

---

## 💻 입출력 예제

| 입력 예시 (Input) | 출력 예시 (Output) |
| :--- | :--- |
| 2<br>4<br>6 4<br>6 3<br>3 1<br>2 0<br>4<br>-5 4<br>2 2<br>-3 1<br>-3 0 | X^4+6X^3+2X^2-1 |
| 3<br>2<br>2 1<br>1 0<br>2<br>-3 1<br>-3 0<br>2<br>1 3<br>2 0 | X^3-X |
| 2<br>1<br>1 0<br>1<br>-1 0 | 0 |

---

## 🔍 예제 설명

* **예제 #1**
  * 입력된 두 다항식은 $6X^4 + 6X^3 + 3X + 2$ 와 $-5X^4 + 2X^2 - 3X - 3$ 입니다.
  * 동류항끼리 합산하면 $(6-5)X^4 + 6X^3 + 2X^2 + (3-3)X + (2-3) = 1X^4 + 6X^3 + 2X^2 + 0X - 1$ 이 됩니다.
  * 출력 규칙에 따라 계수 `1`과 `0X`를 정리하면 **`X^4+6X^3+2X^2-1`**이 됩니다.

* **예제 #2**
  * 입력된 세 다항식의 합은 $(2X + 1) + (-3X - 3) + (1X^3 + 2) = X^3 - X$ 가 되므로 **`X^3-X`**를 출력합니다.

* **예제 #3**
  * $(1) + (-1) = 0$이 되므로 규칙에 따라 **`0`**을 출력합니다.
