# [JUNGOL] 5332 : 교실에서 잠자기1

## 📝 문제 설명
준이는 $N$시간 동안 각 수업 시간마다 $a_i$번 잠을 잤습니다. 현이는 인접한 두 수업 시간의 기록을 합치는(병합) 방식으로 기록을 수정하려고 합니다. 
* **수정 연산:** 인접한 두 값을 더해 하나의 값으로 합침 (예: `[1, 2, 3]`에서 뒤의 두 개를 합치면 `[1, 5]`가 됨)

현이의 목표는 기록에 남은 **모든 숫자를 동일하게 만드는 것**입니다. 모든 숫자를 동일하게 만들기 위해 필요한 **최소 수정(병합) 횟수**를 구하는 프로그램을 작성하시오.

---

### 📥 입력 형식
* 입력의 첫 번째 줄에는 해결할 하위 테스트 케이스의 수인 $T$가 입력된다. ($1 \le T \le 10$)
* 그 다음 줄부터는 $T$번에 걸쳐 각각 두 줄씩 정보가 입력된다.
  * 첫 번째 행은 수업 시간 $N$이 주어진다. ($1 \le N \le 10^5$)
  * 두 번째 행은 각 수업 시간에 준이가 잠든 횟수 $a_1, a_2, \dots, a_N$이 공백으로 구분되어 주어진다. ($0 \le a_i \le 10^6$)
* 각 하위 테스트 케이스에서 모든 숫자의 합이 $10^6$을 초과하지 않음이 보장된다.
* 모든 하위 테스트 케이스의 $N$ 합계가 $10^5$를 초과하지 않음이 보장된다.

### 📤 출력 형식
* 각 하위 테스트 케이스별로 현이가 기록의 모든 숫자를 동일하게 만드는 데 필요한 **최소 수정 횟수**를 줄 바꿈으로 구분하여 출력하시오.

---

## 💻 입출력 예제

| 입력 예시 (Input) | 출력 예시 (Output) |
| :--- | :--- |
| 3<br>6<br>1 2 3 1 1 1<br>3<br>2 2 3<br>5<br>0 0 0 0 0 | 3<br>2<br>0 |

> **힌트/참고 사항**
> * **예제 #1 (`1 2 3 1 1 1`):** 최종 레코드의 모든 숫자가 `3`이 되도록 총 3번 수정할 수 있습니다.
>   * `1 2 3 1 1 1` $\rightarrow$ 앞의 1과 2를 합침 $\rightarrow$ `3 3 1 1 1` (1회)
>   * `3 3 1 1 1` $\rightarrow$ 뒤의 1과 1을 합침 $\rightarrow$ `3 3 2 1` (2회)
>   * `3 3 2 1` $\rightarrow$ 마지막 2와 1을 합침 $\rightarrow$ `3 3 3` (3회) $\rightarrow$ **결과: 3**
> * **예제 #2 (`2 2 3`):** 2번의 수정으로 모든 레코드를 `7`로 변경할 수 있습니다.
>   * `2 2 3` $\rightarrow$ 앞의 2와 2를 합침 $\rightarrow$ `4 3` (1회)
>   * `4 3` $\rightarrow$ 4와 3을 합침 $\rightarrow$ `7` (2회) $\rightarrow$ **결과: 2**
> * **예제 #3 (`0 0 0 0 0`):** 이미 모든 숫자가 `0`으로 일치하므로 아무 작업도 필요 없습니다. $\rightarrow$ **결과: 0**
